# Why This Project Exists

This project exists because learning modern AI often focuses on using tools instead of understanding the ideas behind them.

Large Language Models are frequently presented as black boxes, making it difficult to develop deep intuition or independent thinking.

The purpose of this project is to rediscover every major concept from first principles: understanding why each solution exists, how it works, where its limitations are, and how it can be rebuilt from scratch.

The ultimate goal is not only to learn today's AI systems, but to develop the ability to reason about and design tomorrow's systems.
