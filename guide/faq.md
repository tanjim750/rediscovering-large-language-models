# Frequently Asked Questions (FAQ)

![Project Overview](../assets/all.png)

## What is the goal of this project?

The goal is to develop a deep, first-principles understanding of Large Language Models. Rather than memorizing architectures, libraries, or APIs, this project focuses on understanding why each idea exists, how it works, and how it can be built from scratch.

---

## Who is this project for?

This project is for students, developers, researchers, educators, and anyone curious about how modern AI systems work. Whether you are just starting or already building AI applications, the focus is on developing intuition rather than simply learning tools.

---

## Do I need prior Machine Learning knowledge?

No.

The roadmap starts from computer fundamentals and gradually builds toward modern LLM systems. The only recommended prerequisites are basic Python programming and high-school mathematics.

---

## Is this another prompt engineering or API tutorial?

No.

This project is about understanding the underlying ideas behind Large Language Models. Prompt engineering, APIs, and frameworks are useful, but they are not the primary focus.

---

## Will this project teach me how to build an LLM?

Yes.

The final chapters focus on implementing the major components of an LLM from scratch, including a tokenizer, embeddings, self-attention, transformer blocks, a Tiny GPT, an inference engine, RAG, and a simple AI agent.

---

## Why does the project start with Computer Fundamentals?

Modern AI is built on computers. Understanding memory, processors, parallel computing, and numerical computation provides the foundation for understanding why modern LLMs are designed the way they are.

---

## Why is mathematics included?

Mathematics is the language used to describe machine learning.

This project emphasizes intuition before mathematics. Mathematical concepts are introduced only after understanding the problem they solve.

---

## Why are experiments included?

Reading explanations alone rarely creates intuition.

Experiments help verify ideas, expose limitations, and build confidence through observation rather than memorization.

---

## Why do I need to create my own hypothesis before reading?

Independent thinking is an important part of learning.

Your hypothesis does not need to be correct. Comparing your own reasoning with established knowledge helps develop stronger intuition and better long-term understanding.

---

## Is this project research-oriented?

Yes.

Beyond implementation, the project encourages critical thinking by asking why existing solutions work, where they fail, and how they might be improved.

---

## Which programming language is used?

Python is used for implementations because it is widely adopted in AI research and development. Most implementations begin with simple code and gradually introduce modern libraries where appropriate.

---

## Which frameworks are used?

Frameworks are introduced only when they help explain a concept. The project emphasizes understanding the underlying implementation before relying on high-level abstractions.

---

## Is this project beginner friendly?

Yes.

The roadmap is designed to build knowledge progressively, starting from the fundamentals and gradually introducing more advanced topics.

---

## Can I contribute?

Absolutely.

Contributions are welcome in the form of explanations, diagrams, implementations, experiments, documentation improvements, bug fixes, and research discussions. Please read the **CONTRIBUTING.md** guide before contributing.

---

## How should I study this project?

Follow the project's learning philosophy:

1. Ask a question.
2. Understand the problem.
3. Form your own hypothesis.
4. Learn the theory.
5. Implement it.
6. Experiment with it.
7. Reflect on what you learned.

The goal is not to memorize answers, but to develop a mental model that allows you to reason independently.

---

## What makes this project different from other AI resources?

Most resources teach *how to use* AI.

This project teaches *why AI works*.

Instead of presenting finished solutions, it explores the engineering decisions, trade-offs, and first principles that led to modern Large Language Models.

---

## Will this project always stay up to date?

The field of AI evolves rapidly. While specific models, frameworks, and techniques may change, this project is built around first principles that remain valuable regardless of future advancements. The roadmap and content will continue to evolve as new research and better ideas emerge.
