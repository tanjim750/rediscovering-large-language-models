# Prerequisites

This project is designed to be beginner-friendly while gradually building toward advanced AI systems. You do **not** need prior knowledge of Machine Learning, Deep Learning, or Large Language Models.

However, having the following foundations will help you get the most out of this journey.

![Roadmap](../assets/roadmap.png)

---

# Required Knowledge

## Python Programming

You should be comfortable with basic Python programming, including:

* Variables and data types
* Functions
* Loops and conditional statements
* Lists, dictionaries, and tuples
* Basic object-oriented programming
* Reading and writing files

You do **not** need advanced Python knowledge before starting.

---

## High School Mathematics

The project assumes familiarity with:

* Basic algebra
* Functions
* Exponents
* Graphs
* Basic probability

More advanced mathematical concepts such as vectors, matrices, derivatives, gradients, and linear transformations are introduced from first principles throughout the project.

---

# Helpful (But Not Required)

The following topics are helpful but **not required**:

* Basic Computer Science
* Data Structures
* Algorithms
* Basic Linear Algebra
* Basic Statistics
* Git and GitHub

If you are unfamiliar with any of these topics, don't worry—they are introduced whenever they become necessary.

---

# No Prior AI Experience Required

You do **not** need experience with:

* Machine Learning
* Deep Learning
* Neural Networks
* PyTorch
* TensorFlow
* Transformers
* Prompt Engineering
* LLM Frameworks

The roadmap begins with computer fundamentals and gradually builds every concept from the ground up.

---

# Required Mindset

This project values curiosity more than prior knowledge.

You do not need to know everything before you begin.

Instead, bring the willingness to:

* Ask questions.
* Think independently.
* Form your own hypotheses.
* Build things yourself.
* Learn from experiments.
* Accept being wrong.
* Continuously improve your understanding.

A curious beginner will often learn more than someone who only memorizes definitions.

---

# What You Need

Before starting, make sure you have:

* A computer capable of running Python.
* Python 3.11 or later installed.
* Git installed (recommended).
* A GitHub account (recommended).
* A code editor such as Visual Studio Code.

As the project progresses, additional tools and libraries will be introduced with clear installation instructions.

---

# Time Commitment

There is no fixed schedule.

Everyone learns differently.

Rather than trying to finish quickly, focus on developing a deep understanding of each topic before moving to the next one.

Progress measured by understanding is far more valuable than progress measured by speed.

---

# Learning Expectations

This project is not intended to be consumed passively.

For every topic, you are encouraged to:

* Read carefully.
* Think before looking at the answer.
* Build the implementation.
* Experiment with different ideas.
* Reflect on what you learned.

The goal is not simply to complete the roadmap, but to develop a mental model that allows you to understand, build, and reason about modern AI systems independently.

---

# One Final Recommendation

If you ever feel tempted to memorize a concept, pause and ask yourself:

> **"Do I understand why this exists?"**

If the answer is **yes**, continue.

If the answer is **no**, go back and understand the problem first.

Everything in this project is built on that simple principle.
