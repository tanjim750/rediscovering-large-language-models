# Contributing

Thank you for your interest in contributing to this project.

Contributions of all kinds are welcome, including improvements to explanations, implementations, diagrams, experiments, documentation, bug fixes, and educational resources.

Before submitting a pull request, please discuss significant changes through an issue so that ideas can be reviewed and aligned with the project's learning philosophy.

Every contribution should prioritize clarity, correctness, first-principles thinking, and educational value over unnecessary complexity.
